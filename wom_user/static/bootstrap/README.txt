Bootstrap files

This folder contains CSS, js and font files for bootstrap with the
following mix:

- The base is from:
  https://cdn.jsdelivr.net/npm/bootstrap-icons@1.13.1/font/bootstrap-icons.min.css

- The look and feel is customized with https://bootswatch.com/united/

- The fonts have been copied here (which meant the above style had to
  be modified to avoid pulling from from a third party Web API), from
  https://github.com/canonical/Ubuntu-Sans-fonts/tree/main/fonts

- The bootstrap icons have also been copied here, from
  https://icons.getbootstrap.com/
